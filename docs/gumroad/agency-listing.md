## What You Get

This template pre-configures izhubs ERP specifically for **agency and freelancer workflows**:

### ✅ 8-Stage Sales Pipeline
Purpose-built for client acquisition and retention:

| Stage | Purpose |
|-------|---------|
| **Lead mới** | Incoming inquiry, not yet qualified |
| **Gửi proposal** | Proposal sent, waiting for feedback |
| **Đàm phán** | Active negotiation on price/scope |

No setup required. Fields appear immediately after import:

- **Kênh tiếp cận** — Google / Facebook / Referral / Cold outreach / LinkedIn
- **Quy mô công ty** — 1-10 / 11-50 / 51-200 / 200+
- **Ngành** — free-text industry field per company

## How to Install

1. Download this package
2. Open [izhubs ERP](https://app.izhubs.com) (free account or self-hosted)
3. Go to **Settings → Industry Template → Import from file**
4. Upload `template.json`
5. Done — your CRM is configured in under 60 seconds

A: No. izhubs ERP is open source and free to self-host. This template is the paid setup package — saves you hours of configuration.
