# 🏢 Coworking Space CRM Template for izhubs ERP

> **Instant CRM for coworking space operators, flexible workspace providers, and shared office managers.**
> Manage members, leads, and renewals — all in one place.

---

## What You Get

### ✅ 7-Stage Member Pipeline

| Stage | Purpose |
|-------|---------|
| **Lead** | Enquiry received, not yet toured |
| **Tour Scheduled** | Property tour booked |
| **Proposal Sent** | Pricing package shared |
| **Trial** | Free trial / day pass active |
| **Active Member** | Paying member |
| **Renewal** | Contract expiring soon |
| **Churned** | Membership ended |

### ✅ Custom Fields for Coworking
- **Package Type** — Hot desk / Dedicated desk / Private office / Virtual office
- **Contract Duration** — Monthly / Quarterly / Annual
- **Team Size** — 1 / 2-5 / 6-10 / 10+
- **Move-in Date** — expected start date
- **Budget (VND/month)** — willingness to pay
- **Acquisition Source** — Walk-in / Google / Instagram / Referral / Event

### ✅ 2 Pre-built Automations
- **Auto tour follow-up** → reminder 1 day after tour scheduled
- **Renewal alert** → task created 30 days before contract end (Renewal stage)

### ✅ Dashboard Layout
- Member count snapshot
- Revenue this month
- Occupancy by package type
- Upcoming renewals

### ✅ Sample Data
- 5 sample members (startup founders, freelancers, SME teams)
- 3 sample deals across different pipeline stages

## How to Install

1. Download this package
2. Open [izhubs ERP](https://app.izhubs.com)
3. Go to **Settings → Industry Template → Import from file**
4. Upload `template.json`
5. Import `sample-contacts.csv` via the Import module
6. You're live in under 5 minutes
