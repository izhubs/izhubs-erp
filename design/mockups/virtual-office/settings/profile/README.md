# Settings — Cài đặt

> **Loại file:** Mockup UI (Google Stitch, Gemini 3 Pro)  
> **Stitch Screen ID:** `268fa81037704a3bb0525915b0bfdd8d`

## Nghiệp vụ quản lý
Cài đặt tài khoản cá nhân và workspace. Quản lý profile, bảo mật, thành viên nhóm, billing, thông báo, và API keys.

## Vì sao có màn hình này?
Mọi SaaS cần settings. Người dùng cần tự chỉnh thông tin cá nhân, Admin cần quản lý billing và thành viên nhóm mà không cần liên hệ support.

## Vấn đề giải quyết
- Admin muốn thêm/xóa thành viên team mà không cần dev can thiệp
- Người dùng muốn đổi avatar hoặc cập nhật chức danh
- Quản lý API Keys cho integrations bên thứ 3

## Mong muốn
- Secondary nav bên trái → điều hướng nhanh giữa các mục settings
- Profile section là default → phổ biến nhất
- Danger Zone rõ ràng, không dễ bấm nhầm
